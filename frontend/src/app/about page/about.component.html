About Keenan's Kash App

This web application is an online bank account app which allows new and existing users to deposite and withdraw funds from their account. This is a MEAN stack application which was made with the use of many libraries such as Bootstrap and Material for the css and design aspect of the website, and Passport for the login account authenticaion.
